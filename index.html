Medical Imaging Multimodality Breast Cancer Diagnosis User Interface

Research Assistance Project

The Medical Imaging Multimodality Breast Cancer Diagnosis User Interface (MIMBCD-UI) project deals with the use of a recently proposed technique in literature: Deep Convolutional Neural Networks (CNNs). These deep networks will incorporate information from several different modes: Magnetic Resonance Imaging (MRI) volumes, UltraSound (US) images, MammoGraphic (MG) images (both views CC and MLO) and text. Therefore, the design and development of an User Interface (UI) is necessary, allowing the user (in this case, the medical specialist) to display various types of image (i.e., MG, US and MRI).

Medical Imaging Diagnosis Assistant

Research Assistance Project

The Medical Imaging Diagnosis Assistant (MIDA) project joins contributions from the MIMBCD-UI project. The MIDA project deals with the development and integration of several AI-Assisted techniques across an User Interface (UI). It aims to approach several areas of the Computer Science field. The areas are Artificial Intelligence (AI), Machine Learning (ML) and eXplainable Artificial Intelligence (XAI). The main goal of the project is to develop an Assistant to help the automation of the cancer diagnosis regarding the use of Medical Imaging (MI) technologies.

BreastScreening

Research Project

The BreastScreening project joins contributions from both MIMBCD-UI and MIDA projects. The BreastScreening aims at developing of an Assistant for the breast cancer diagnosis. We use several modalities of medical images. The modalities are Magnetic Resonance Imaging (MRI) volumes, UltraSound (US) images and MammoGraphic (MG) images (both views CC and MLO). From here, with the obtained data, we can create an algorithm to consume the data.
